
Brainstorm Potential Elements/Features:
•	Change the background color when click on the teammates picture
•	Set link to github profile
•	Ability to leave comments to user profile

Convert Ideas into User Stories:
•	As a teammate, I want to see personalized profile page so that I would like to use my favorite color.
•	As a customer, I want to know more about team so that I would see their previous projects.
•	As a teamlead, I want to make motivating comments for my team so that I can do them with this feature. 

Determine the MVP:
•	Clickable team name
•	Team member webpage on a different page
•	Title on the main page
Categorize features as:
•	Must Have:  Clickable team name, Team member webpage on a different page, Title on the main page
•	Nice to Have: Link to the GitHub account
•	Could Have: Ability to leave messages or comments
•	Will Not Have (Out of Scope): Animated header
 Constraints and Assumptions
•	Time limit
•	Knowledge limit
•	What is the color code?
•	What features could the project has?
•	Who need access to the team list? Authentication?
