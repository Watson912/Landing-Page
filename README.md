\# Markdown Reference \& Tutorial

Welcome! This is a \*\*complete guide to Markdown\*\*, the lightweight markup language used for formatting text on GitHub, in README files, and more.

It includes explanations, examples, end exercises so you can \*\*learn by doing\*\*.
